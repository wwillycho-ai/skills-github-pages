# GitHub Pages

_GitHub Pages를 사용하여 GitHub 저장소에서 사이트나 블로그를 만들어 보세요._

## 환영합니다

- **대상**: 초보자, 학생, 프로젝트 관리자, 소규모 비즈니스.
- **배우게 될 내용**: GitHub Pages 사이트를 구축하는 방법.
- **만들게 될 것**: 블로그가 포함된 간단한 GitHub Pages 사이트를 만듭니다. 정적 사이트 생성기인 [Jekyll](https://jekyllrb.com)을 사용합니다.
- **사전 준비**: branch, commit, pull request에 대해 배워야 한다면 먼저 [GitHub 소개](https://github.com/skills-kr/introduction-to-github) 과정을 수강하세요.
- **소요 시간**: 이 실습은 완료하는 데 1시간 미만이 소요됩니다.

이 실습에서 다음을 수행합니다:

1. GitHub Pages 활성화
1. 사이트 구성
1. 홈페이지 커스터마이징
1. 블로그 게시물 작성
1. Pull request 병합
